Relative Velocity Problems: When relative velocities are involved, use the following steps:
  1. List the given quantities in a form of notation that allows you to recognize which object is moving in what reference frame.
  2. Draw a diagram showing the relationship between the vector velocities of the problem.
  3. Write the equations relating the horizontal and vertical components of these vectors.
  4. Solve the equations simultaneously for the unknowns of interest.
