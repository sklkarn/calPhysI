Solution 5.3.4.1.

\(F = 1000 N, \quad m = 100 kg, \quad x = 2 m, \quad \mu = 0.2,\quad v_{o} = 0.\) Equations:
\begin{equation*} \sum F_{x} = F - f = ma; \quad \sum F_{y} = N - w = 0; \quad w = mg= 980 N = N; \end{equation*}
\begin{equation*} f = \mu N= 196 N \end{equation*}
  1. \begin{equation*} W_{F} = \int \vec{F} \cdot \vec{\,dr} = \int F \,dr = F x = (1000 N)(2 m) = 2000 J \end{equation*}
  2. \begin{equation*} W_{g} = \int \vec{w}\cdot\vec{\,dr} = \int w \,dr\cos\theta = w x \cos(90^{o}) = 0 J \end{equation*}
  3. \begin{equation*} W_{f} = \int \vec{f}\cdot\vec{\,dr} = \int f \,d r \cos\theta = f x \cos(180^{o}) = - f x = -392 J \end{equation*}
  4. \begin{equation*} v^{2} = v^{2}_{o} + 2 a x;\quad a = 8.04 m/s^{2}; \quad \therefore v = 5.67 m/s \end{equation*}
